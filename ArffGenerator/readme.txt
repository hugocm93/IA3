Projeto para a Disciplina de Inteligência Artificial, na PUC-Rio

Esse projeto java tem o propósito de gerar o arquivo .arff. Este arquivo serve de entrada no Weka, para realizar o aprendizado de máquina. Dentro das pastas part1 e part2 estão as reviews dos filmes. O arquivo ListOfWords.txt é um arquivo auxiliar gerado pelo programa. São gerados dois arquivos arff:
MovieDataSet.arff e MovieDataSet_REDUCED.arff. O segundo é gerado a partir do primeiro pelo weka. O weka filtra automaticamente os atributos mais relevantes. Com o arquivo .arff pronto, é possível executá-lo no weka. Atenção para a quantidade de tempo necessária para rodar no dataset! Demora muito. O ideal é deixar rodando durante a noite. Cada um de nós poderia rodar um algoritmo em sua máquina.
